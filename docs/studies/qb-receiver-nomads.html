What this study is trying to catch

We are looking for QB-to-receiver chemistry that survives context changes. Instead of a single team snapshot, we search for duos that reappear across multiple team-seasons, especially when they span multiple leagues.

How we define a combo

For each team-season, we pair every QB with every WR/RB/TE/FB/HB teammate. We then score that team-season pairing using the QB's passing production and the receiver's receiving production from the same league-season bucket.

When the same QB-receiver pair shows up again in another team-season, we treat it as recurring chemistry and add those scores together.

OL-linked units

For each recurring QB-receiver duo, we also look for offensive linemen who repeatedly show up in those same team-seasons. This surfaces additional unit-level links: not just who caught passes, but who may have protected that connection.

Methodology

Data source: player JSON files in docs/data/players/ plus season metadata from docs/data/sports.json.

A team-season is identified by (sport_id, team) roster membership. Because many feeds do not include direct passer-target links, this study uses shared team-season co-membership as a proxy for QB-receiver connection.

Chemistry score per team-season: QB score + Receiver score, where QB score = passing_yards + 40*passing_tds - 45*interceptions and Receiver score = receiving_yards + 60*receiving_tds + 2*targets.

Scores are descriptive and ranking-oriented, not causal estimates. This study reruns on each build as new rosters and statlines land.
